# Actividad de Aprendizaje Trabajo 1

## Maquetación de una Página Web con Tablas
Página web creada exclusivamente con HTML utilizando principalmente tablas para ordenar los elementos que la conforman.

### Sitio Web
[[Ferroguate.com](http://intecap-william-tareas.atwebpages.com/aat1_william_pereira)](http://intecap-william-tareas.atwebpages.com/aat1_william_pereira)

### Fecha 
06 de Marzo de 2026

### Descripción
Página web que muestra como se creaban las páginas sin el uso de técnicas modernas y que simula la página de una ferretería.

#### Página de Inicio
La página de inicio muestra en la parte superior un Banner con el logotipo y la información de la empresa. Un menu de navegación a otras páginas
En la parte principal.

En el cuerpo se muestran imagenes con las diferentes tipos de ofertas del mes.

En la parte inferior se muestra un pie de página con la información el pbx y la dirección exacta de la empresa y autor de la página.

##### inicio.html
![página de inicio](/imagenes/pagina-inicio.png)

#### Página de Productos y Servicios
Se muestra en cuatro columnas la información de los productos, cada producto muestra una imagen del producto, su nombre, precio y un boton "mas información" para ver la información especifica de ese producto.

##### productos-y-servicios.html
![página de inicio](/imagenes/pagina-productos.png)

#### Nuestras Tiendas
Esta página muestra un mapa con la ubicación de las tiendas (2 tiendas).

##### tiendas.html
![página de inicio](/imagenes/página-tiendas.png)

#### Registrarse
Muestra un formulario en donde los clientes pueden registrarse, se utilizaron diferentes elementos de html y se ordeno la información en una tabla.

##### registrarse.html
![página de inicio](/imagenes/página-registro.png)
